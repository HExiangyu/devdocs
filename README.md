# Building this site

你可以使用Jekyll创建这个文档网站. 
创建方法如下:

```bash
# Copy the sample config file and name it "config.yml"
$ cp _config.devdocs.yml _config.yml

# Install dependencies
$ bundle install

# Visit http://localhost:4000 in your favorite browser!
$ bin/jekyll serve
```
